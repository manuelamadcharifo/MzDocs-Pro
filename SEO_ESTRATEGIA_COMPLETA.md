## 4. ESTRATÉGIA DE INTERLINKING

### 4.1 Regras Base
```
CADA página SEO deve ter:
  ✓ 1 link para a homepage (/)
  ✓ 1 link para a página de geração do documento relevante (#gerar)
  ✓ 2–4 links para artigos do mesmo cluster
  ✓ 1 link para artigo de cluster diferente (cross-cluster)
  ✓ CTA interno com anchor text transaccional
```

### 4.2 Mapa de Interlinking por Cluster

**Cluster CV → aponta para:**
- `como-fazer-cv` ↔ `cv-licenciado` ↔ `cv-sem-experiencia`
- `como-fazer-cv` → `carta-candidatura` (cross: carreira)
- `cv-*` → `/` com anchor "criar CV profissional grátis"

**Cluster Jurídico → aponta para:**
- `procuracao` ↔ `contrato-arrendamento` ↔ `declaracao-residencia`
- `contrato-arrendamento` → `requerimento` (cross: documentos)
- Todos → `/` com anchor "gerar contrato online"

**Cluster Académico → aponta para:**
- `trabalho-escolar` ↔ `normas-academico` ↔ `como-citar`
- `trabalho-escolar` → `como-fazer-cv` (cross: carreira — "após concluir a licenciatura")

### 4.3 Anchor Texts Aprovados (evitar genérico "clique aqui")

| Contexto | Anchor Text Correcto |
|---|---|
| Link para homepage | "criar {documento} online em Moçambique" |
| Link entre artigos CV | "modelo de CV em Moçambique" |
| Link para requerimento | "fazer requerimento em Moçambique" |
| Link para contratos | "contrato de arrendamento moçambique" |
| CTA geração | "Gerar o Meu {Documento} Agora" |

### 4.4 Hub Page `/pages/index.html`
A página índice deve funcionar como **hub de categoria** com:
- H1: "Guias e Modelos de Documentos em Moçambique"
- Listagem por cluster com H2 por categoria
- Cada artigo com snippet de 30 palavras + link
- Schema `CollectionPage` + `BreadcrumbList`

## 6. ESTRATÉGIA GOOGLE DISCOVER

O Google Discover mostra conteúdo mesmo sem pesquisa activa. Para aparecer:

### 6.1 Requisitos Técnicos
```
✓ Core Web Vitals verdes (LCP < 2.5s, INP < 200ms, CLS < 0.1)
✓ Imagem large (mín. 1200×628px) em cada página — og:image
✓ HTTPS activo
✓ Indexação confirmada no Google Search Console
✓ Mobile-first (PWA já activa — vantagem)
```

### 6.2 Tipos de Conteúdo que Funcionam no Discover

**Formato A — Estacional / Oportunidade:**
> "Época de Candidaturas na Função Pública: Como Preparar o Seu Dossier"  
> Publicar em Janeiro/Julho (períodos de concursos públicos em MZ)

**Formato B — Lista Prática:**
> "7 Erros no CV que Fazem os Recrutadores Moçambicanos Rejeitar"  
> "5 Documentos que Todo Moçambicano Deve Ter Prontos"

**Formato C — Actualidade + Utilidade:**
> "Novas Regras de Arrendamento em Moçambique 2025: O Que Muda no Contrato"

**Formato D — Histórico de Sucesso:**
> "Como Consegui Emprego no BCI com Este CV — História Real"

### 6.3 Calendário Editorial para Discover

| Mês | Tema | Gatilho |
|---|---|---|
| Janeiro | Candidaturas emprego início de ano | Temporada de contratações |
| Março | Trabalhos escolares 1.º trimestre | Época académica |
| Junho | Candidaturas função pública | Concursos públicos MZ |

| Novembro | Trabalhos de culminação | Defesas de monografias |

### 6.4 Imagens OG (Obrigatório para Discover)
- Dimensão mínima: **1200 × 628px**
- Formato: JPG ou WebP
- Texto na imagem (máx 20% de área)
- Estilo consistente com identidade MzDocs (fundo escuro + verde #009A44)
- Naming: `/assets/og/{slug}.jpg`
